## 🔧 Cập Nhật SEO Trong Tương Lai

### Thêm từ khóa mới:
Edit file: `src/config/seo.ts`

### Cập nhật meta cho trang:
```tsx
import SEO from '../components/SEO'

<SEO 
  title="Tiêu đề mới"
  description="Mô tả mới"
  keywords="từ khóa 1, từ khóa 2"
/>
```

### Cập nhật sitemap:
Edit file: `public/sitemap.xml`
